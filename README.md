# 3DSoundShell

Français:

3DSoundShell est un lecteur de son pour 2/3DS basé sur le célèbre explorateur de fichier 3DShell comme son nom l'indique, le code a était fait par [Claude.ai](https://claude.ai/new) et les images / son de l'app on était fait par moi. L'application n'ai pas encore fini il y a des bugs a corrigé mais elle peux deja lire les musiques avec une joli interface.

\
English: 

3DSoundShell is a music player for the 2DS and 3DS based on the popular 3DShell file explorer. As the name suggests, the code was written by [Claude.ai](https://claude.ai/new), and I created the app’s graphics and sound. The app isn’t finished yet—there are still some bugs to fix—but it can already play music with a nice interface.
